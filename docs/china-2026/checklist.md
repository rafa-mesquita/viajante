# Checklist — Antes de embarcar

## Documentos

- [ ] Passaporte válido por +6 meses depois da volta
- [ ] Cópia digital do passaporte (Drive + email)
- [ ] Comprovante de hospedagem impresso (1ª noite — pedem na imigração)
- [ ] Passagem ida e volta impressa
- [ ] Seguro viagem com cobertura pra China (mínimo USD 30k)
- [ ] Comprovante vacinação COVID (caso voltem a exigir)
- [ ] Carteira de motorista internacional (se for dirigir — improvável)
- [ ] Foto 5x7 fundo branco (pra emergências)

## Apps a instalar ANTES (alguns só funcionam fora da China)

- [ ] **VPN** (ExpressVPN/Astrill/Nord) — instalar e testar antes
- [ ] **WeChat** + criar conta (precisa de outro usuário pra "verificar")
- [ ] **Alipay** — adicionar cartão internacional
- [ ] **Pleco** (dicionário chinês offline)
- [ ] **Google Translate** + baixar pacote chinês offline
- [ ] **Trip.com** (trens, voos internos, hotéis)
- [ ] **Didi** (Uber chinês)
- [ ] **Amap** ou **Baidu Maps**
- [ ] **DiDi Hitch** (carona, opcional)
- [ ] **Mapas offline** Google Maps de cada cidade (caso a VPN falhe)

## Bagagem — frio extremo

### Roupas
- [ ] Casaco puffer/down jacket pesado (essencial pra Pequim)
- [ ] 2-3 camisetas térmicas (segunda pele superior)
- [ ] 2 calças térmicas (segunda pele inferior)
- [ ] Calças (calça térmica + jeans/social por cima)
- [ ] 5-6 camisetas + 3 blusas de manga longa
- [ ] Suéter/moletom
- [ ] Meias térmicas (5+ pares)
- [ ] Cuecas/calcinhas (10+)
- [ ] Gorro
- [ ] Cachecol/bandana
- [ ] Luvas (touchscreen é melhor)
- [ ] Botas confortáveis impermeáveis (vai andar 15-20 km/dia)
- [ ] 1 par tênis backup
- [ ] Chinelo de hotel (alguns têm, alguns não)
- [ ] Pijama quente

### Eletrônicos
- [ ] Adaptador universal de tomadas (China usa A/C/I)
- [ ] Power bank (até 100Wh, ou levam embora no aeroporto)
- [ ] Carregador celular + cabo extra
- [ ] Carregador notebook (se for levar)
- [ ] Câmera + cartão SD + bateria extra
- [ ] Fones (conversão USB-C se for o caso)

### Saúde
- [ ] Remédios pessoais
- [ ] Anti-diarreico (Imodium)
- [ ] Antialérgico
- [ ] Anti-inflamatório (Ibuprofeno)
- [ ] Soro pra hidratação oral (poluição resseca)
- [ ] Hidratante labial e corporal (frio seco racha tudo)
- [ ] Filtro solar (sol no inverno chinês ainda queima)
- [ ] Máscara N95 (poluição em Pequim em dias ruins)
- [ ] Lenço umedecido + álcool gel

### Diversos
- [ ] Mochila de uso diário (20-25L)
- [ ] Garrafa térmica (chineses adoram água quente, têm em todo lugar pra encher)
- [ ] Cadeado pequeno (locker de hostel)
- [ ] Pochete antifurto (pra passaporte)
- [ ] Sacolas plásticas/zip lock (úteis)
- [ ] Pequeno presente brasileiro (caso conheça alguém)

## Financeiro

- [ ] Avisar banco/cartão sobre viagem
- [ ] 2 cartões de crédito internacionais (1 backup) — preferência Visa/Master
- [ ] Cartão pré-pago em USD (Wise, C6, Nomad) — alguns lugares aceitam
- [ ] CNY em espécie: ¥1.000-2.000 (~R$ 800-1.500) pra emergência
- [ ] Adicionar cartão internacional ao Alipay e WeChat Pay ANTES de viajar

## 1 semana antes

- [ ] Confirmar todas as reservas (hotéis, trens, voos)
- [ ] Imprimir tudo + salvar em PDF na nuvem
- [ ] Compartilhar roteiro com família
- [ ] Verificar previsão do tempo nas cidades
- [ ] Carregar todos os eletrônicos
- [ ] Trocar dinheiro (ou já estar com Wise/C6 carregado em USD)
- [ ] Configurar roaming/eSIM
- [ ] Avisar trabalho/contatos sobre indisponibilidade

## Dia antes

- [ ] Check-in online voo
- [ ] Confirmar transporte CNF/casa → aeroporto
- [ ] Repassar mala (peso, líquidos, bateria na mão)
